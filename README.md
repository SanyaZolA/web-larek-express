WEB-Larek

Ларек интересных вещей.

## Описание

Список WEB-скиллов которые можно положить в корзину, оформить заказ, выбрать способ оплаты, адрес доставки, email и номер телефона. 

## Стек технологий

- Node.js
- Express.js
- MongoDB
- Mongoose
- Typescript
- Celebrate
- Cors
- Dotenv
- и д.р.
